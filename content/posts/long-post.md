# Introduction

Markdown is a lightweight markup language used for writing formatted text using plain text.

It is widely used in documentation, blogs, and static site generators like Hugo.

---

# Headings

You can create headings using `#`:

# H1
## H2
### H3
#### H4
##### H5
###### H6

---

# Bold, Italic, and Strikethrough

- **Bold text**
- *Italic text*
- ***Bold + Italic***
- ~~Strikethrough~~
